# 宿舍管理系统 (C 语言)

一个基于数据结构课程要求实现的命令行宿舍管理系统，采用顺序表存储，支持信息增删改、三种查找算法与三种排序算法。配套课程报告见 `报告.md`。

## 功能概述
- 信息维护：添加、删除、修改、列表展示
- 查找算法：顺序查找、二分查找、哈希查找（开放定址双重散列）
- 排序算法：插入排序(按ID)、快速排序(按姓名)、冒泡排序(按宿舍)，支持升序/降序
- 交互方式：菜单式命令行

## 目录结构
- `include/dorm.h` 头文件，数据结构与函数声明
- `src/dorm.c` 算法实现：增删改、打印、查找、排序
- `src/main.c` 程序入口与菜单交互
- `报告.md` 课程报告文档

## 环境要求
- C 编译器：`gcc` 或 `clang`（亦可使用 MSVC `cl`）
- 终端编码建议 UTF-8；Windows 终端可执行 `chcp 65001` 获得更好的中文显示

## 编译与运行
- 使用 `gcc`：
  - `gcc -std=c11 -O2 -o dorm.exe src/main.c src/dorm.c`
  - 运行：`./dorm.exe`
- 使用 MSVC `cl`：
  - `cl /Fe:dorm.exe src\main.c src\dorm.c`
  - 运行：`dorm.exe`

## 使用说明
- 启动程序后，按提示选择菜单项：
  - `1` 添加，`2` 删除，`3` 修改，`4` 列表，`5` 查找，`6` 排序，`0` 退出
- 二分查找依赖“按ID排序”后的有序表，可先执行排序菜单中的“插入排序(按ID)”再进行二分查找
- 哈希查找会根据当前表动态构建哈希表以进行查询

## 算法说明（简述）
- 顺序表：动态扩容，`O(1)` 追加，删除采用后移覆盖
- 查找：
  - 顺序查找 `O(n)`
  - 二分查找 `O(log n)`（前提：按 ID 有序）
  - 哈希查找 平均近似 `O(1)`
- 排序：
  - 插入排序 稳定，适合近乎有序数据
  - 快速排序 分治，平均 `O(n log n)`
  - 冒泡排序 稳定，教学友好
